# Vue Bootstrap Boilerplate

![logo](https://repository-images.githubusercontent.com/217154004/2402a680-b66f-11ea-9125-22625960bc03)

- Vue.js 3
- Bootstrap 4
- Vuex
- Vue-Router
- Sass/Scss
- ESLint
- Axios

## Getting Started

Clone project and enter in folder:

```
cd vue-bootstrap-boilerplate
```

Run npm install:

```
npm i
```

Enjoy:

```
npm run serve
```
